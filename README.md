# 🏎️ LCR — Frontend

Frontend de **Limando Carter Rfactor**, una plataforma para compartir y gestionar mods de simuladores de carreras (rFactor, Assetto Corsa).

Construido con React + Vite, consume la [API Rest en Spring Boot](https://github.com/Facus006/Api-Lcr).

## 🛠️ Stack

- React 18 + Vite
- React Router DOM v6
- React Bootstrap 5
- JWT (jwt-decode)

## 🚀 Correr localmente

```bash
npm install
npm run dev
```

La app corre en `http://localhost:5173` y consume el backend en `http://localhost:8080`.

## 🔗 Repositorio del backend

👉 [Api-Lcr](https://github.com/Facus006/Api-Lcr)
